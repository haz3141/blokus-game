# Cornerfall Rules Summary

Cornerfall is an original corner-placement polyomino strategy game for 2 to 4 players.

## Core Rules

- each player has a full set of unique polyomino pieces
- the first move must cover the assigned start corner
- later moves must touch the same player's existing pieces only at corners
- later moves must not share an edge with the same player's existing pieces
- pieces cannot overlap or go out of bounds
- players place one piece per turn or pass when no legal move remains
- the game ends when no player can make another legal move
- score is based on remaining unplayed squares with a completion bonus
